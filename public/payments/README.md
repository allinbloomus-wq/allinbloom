# Payment Icons

Place official brand SVG logos in this folder using the filenames below:

- visa.svg
- mastercard.svg
- amex.svg
- discover.svg
- diners-club.svg
- jcb.svg
- unionpay.svg
- apple-pay.svg
- google-pay.svg
- samsung-pay.svg
- amazon-pay.svg
- link.svg
- cash-app-pay.svg

These icons are used in `src/components/cart-view.tsx`.
If a local file is missing, the UI falls back to the current remote icon URLs.

Official brand assets are typically provided in each brand's guidelines or toolkit.
Examples:
- Apple Pay: https://developer.apple.com/apple-pay/marketing/
- Google Pay: https://developers.google.com/pay/api/web/guides/brand-guidelines
- Samsung Pay: https://developer.samsung.com/pay/downloads-and-resources/branding-guidelines.html
